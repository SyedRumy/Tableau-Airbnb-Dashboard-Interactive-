# Airbnb Data Visualization Dashboard (Tableau)

## Project Overview

This project analyzes Airbnb listing data using Tableau to identify trends in pricing, location, and listing availability.
The dashboard allows users to explore data interactively and gain insights into the Airbnb market.

## Objectives

* Analyze average Airbnb prices across neighborhoods
* Identify areas with the highest number of listings
* Visualize price distribution and trends
* Create an interactive dashboard for data exploration

## Tools & Technologies

* Tableau
* Excel
* Data Visualization

## Dashboard Features

* Interactive filters for location and price range
* Visual comparison of listing prices by area
* Distribution of listings across neighborhoods
* Dynamic charts and maps for better insights
